# StudentDormMIS
### 学生宿舍管理系统
* 这是使用`C#`来创建的管理系统，而使用到的工具有：
  * Visual Studio 2019
  * SQL Server 2008 R2
* 系统的功能的模块如下：
  * 基本信息模块
  * 学生入住模块
  * 水电收费模块
  * 卫生检查模块
  * 房屋保修模块
  * 外来人员登记模块
